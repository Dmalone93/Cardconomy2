Three Personas

Buyers & Collectors

Discover, purchase, track, and collect cards. Portfolio management, watchlists, and buylists.

Dashboard: Portfolio value, needs attention, watchlist movements, buylist matches, recent orders

Individual Sellers

List cards, manage offers, ship orders, track earnings. Single and bulk listing flows.

Dashboard: Balance, active listings, pending offers, to ship, activity feed, collection + buylist

Local Game Shops

Manage walk-in submissions, bulk intake, buylist performance, and daily revenue.

Dashboard: Revenue, submissions pending, bulk lots, buylist hits, submission queue, buylist performance

How to Switch Persona

Mobile: You tab → gear icon → Account type → tap to cycle (buyer → seller → store)
Desktop: Account page → same cycling
Onboarding: First visit shows account type selector
Storage: localStorage cc_acct — persists across sessions

UX Issues

Fixed
"Add to my wants" renamed
Renamed to "Add to want list" with secondary outline style so it no longer looks like a buy button.
Fixed
Buylist % buttons now show preview
CurrencyInput presets now display the calculated price (e.g. £12.40) below the percentage label.
Fixed
Inconsistent quantity selectors
Shared QtyInput, CurrencyInput, and SliderInput components now standardise all inputs.
Fixed
Search had no empty state
Empty state with "clear filters" button now shown when no results match.
Fixed
Back to top on long pages
Floating scroll-to-top button appears on product, listing, and search pages after scrolling 300px.

Missing Features

Fixed
Saved/recent searches
Recent searches section with clock icons shown above suggestions, with clear button.
Fixed
Price alerts
Bell icon on watched cards toggles inline alert input. Set a target price and get notified.
Fixed
Return/dispute flow
DisputeScreen with issue type selection, description, photo upload, resolution preference, and claim submission.
Fixed
Shipping label generation
ShippingScreen with item selection, shipping method, package size, and mock label preview with print/download.
Remaining

GBP only. Currency conversion requires deeper architectural changes (locale detection, exchange rates).

Demo Tips

• Clear localStorage to reset all state and re-trigger onboarding
• Switch account type via Settings to demo each flow without clearing data
• Scan card works from sell, search, and collection — different primary actions per context
• LiveSweep scanner accessible from sell-to-shop and bulk listing flows
• Trade builder shows real-time fairness % as you select cards
• Price history charts appear on listings with mock historical data
• Offer thread on listings shows mock negotiation with counter-offers
